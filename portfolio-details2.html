Hoop Central

In this project, being an avid basketball fan, i decided to create a website that shows nba player statistics upon a simple search. To do this i implemented an player statistics API and created a JSON fetch using javascript. I interpolated the user input from a form and inserted it into the api url query paramaters. This allows a fetch of the player inputed into the search bar to be made from the api, in turn, showing their stats. I also did this same process within a youtube api in which returns the players highlights under their statistics.
